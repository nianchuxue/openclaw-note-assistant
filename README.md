# 📝 AI 笔记总结助手

> 基于 OpenClaw 架构 + DeepSeek 大模型的 AI 笔记总结工具

## 这是什么

一个纯前端、零部署的 AI 笔记总结助手。

你贴进去一段笔记，它自动提炼出**关键词、核心观点、详细摘要、思考题**四个模块，一键复制、导出 PDF，历史记录自动保存。

## 快速开始

### 在线使用

1. 下载 `index.html` 到本地
2. 双击用浏览器打开
3. 填入你的 [DeepSeek API Key](https://platform.deepseek.com/api_keys)
4. 粘贴笔记，点击「开始总结」


## 功能

- 📄 **粘贴即总结** — 左侧粘贴笔记，右侧自动输出四模块结果
- 🔑 **关键词提取** — 提取 3-5 个核心关键词
- 🎯 **核心观点** — 一句话概括笔记精髓
- 📖 **详细摘要** — 分点列出要点
- ❓ **思考题** — AI 生成启发性问题，促进深度理解
- 📋 **一键复制** — 每个模块独立复制
- 📄 **导出 PDF** — 生成格式化 PDF 报告
- 📜 **历史记录** — 自动保存，随时回溯
- 🗑️ **清空/重置** — 灵活管理数据

## 技术架构

```
┌─────────────┐    ┌──────────────┐    ┌──────────────┐
│  用户浏览器  │ →  │  DeepSeek API │ →  │  结构化输出  │
│  (纯前端)   │    │  (大模型)    │    │  四模块渲染  │
└─────────────┘    └──────────────┘    └──────────────┘
```

- **架构基础**：基于 OpenClaw 框架的 Agent 设计理念
- **大模型**：DeepSeek-chat (API 调用)
- **前端**：纯 HTML + CSS + JavaScript，单文件无依赖
- **PDF 导出**：jsPDF + html2canvas
- **数据存储**：浏览器 localStorage

## 获取 API Key

1. 访问 [DeepSeek Platform](https://platform.deepseek.com/api_keys)
2. 注册/登录
3. 创建 API Key
4. 复制 Key（以 `sk-` 开头）
5. 粘贴到页面上方的输入框

> 💡 API Key 保存在你本地浏览器中，不会上传到任何服务器
